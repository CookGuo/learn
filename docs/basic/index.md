# 基础篇

前端基础是构建一切高级技能的根基。本模块系统讲解 HTML、CSS 和 JavaScript 的核心知识。

## 内容结构

### HTML
- [语义化与无障碍](./html-semantic.md) - 编写有意义的标记
- [HTML5 新特性](./html5.md) - 现代 Web 开发必备

### CSS
- [BFC 与格式化上下文](./css-bfc.md) - 深入理解布局
- [Flexbox 布局](./css-flexbox.md) - 一维布局解决方案
- [Grid 布局](./css-grid.md) - 二维布局系统

### JavaScript
- [原型与原型链](./js-prototype.md) - 理解 JS 继承机制
- [Promise 与异步](./js-async.md) - 现代异步编程
- [ES6+ 新特性](./js-es6.md) - 语法糖与最佳实践

## 学习建议

::: tip 初学者指南
1. 按照 HTML → CSS → JavaScript 的顺序学习
2. 每学完一个知识点，立即动手实践
3. 使用 [MDN](https://developer.mozilla.org/) 作为参考文档
4. 在 [CodePen](https://codepen.io/) 上练习代码片段
:::
